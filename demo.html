Quick HTML/CSS Setup

Integrating standalone liquid glass in your HTML sites

Simply add the HSL styling variables in your CSS, and drop the SVG `` block at the bottom of your HTML body. No complex libraries or dependencies required.

<!-- 1. Include HSL styling tokens in your header stylesheets --> <style> .my-panel { background-color: rgba(255, 255, 255, 0.12); backdrop-filter: blur(14px) url(#liquid-glass-filter) saturate(160%); box-shadow: inset 0 0 0 1px rgba(255, 255, 255, 0.2), 0 4px 12px rgba(0,0,0,0.3); } </style> <!-- 2. Inject this SVG node anywhere at the bottom of your document body --> <svg aria-hidden="true" style="position:absolute; width:0; height:0; pointer-events:none;"> <filter id="liquid-glass-filter" primitiveUnits="objectBoundingBox"> <feImage width="100%" height="100%" href="data:image/webp;base64,..." /> <feGaussianBlur stdDeviation="0.02" /> <feDisplacementMap in="blur" scale="0.5" xChannelSelector="R" yChannelSelector="G" /> </filter> </svg>
